# 🎮 Hungama Game

**Hungama Game** is a simple text-based **Hangman Game** built using **Python**. This project was developed as **Task 1** for the **CodeAlpha Python Programming Internship**.

The game randomly selects a word from a predefined list, and the player must guess the hidden word one letter at a time. The player has **6 chances** to make incorrect guesses before the game ends.

---

## ✨ Features

- 🎲 Random word selection
- 🔤 Guess one letter at a time
- 📝 Five predefined words
- ❌ Maximum of 6 incorrect guesses
- ✅ Input validation
- 🚫 Prevents duplicate guesses
- 🎉 Win and Game Over messages
- 🔄 Play Again option
- 💻 Console-based interface

## 🎮 How to Play

1. Start the game.
2. Guess one letter at a time.
3. If the guessed letter is correct, it is revealed in the word.
4. If the guess is incorrect, you lose one chance.
5. You have **6 incorrect guesses**.
6. Guess the complete word before your chances run out to win.
